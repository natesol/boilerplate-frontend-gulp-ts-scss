Basic Front-end Boilerplate!

Clean project template for a frontend webapp for HTML, SCSS and TypeScript

This is a fully featured project starter for front-end web application. This project is configured using Gulp 4 to process HTML, SCSS, TypeScript and to optimize images (jpg/png files).


In this proejct, Gulp is configured to run the following functions:
  • minify any HTML files
  • compile any SCSS files to a CSS bundle file
  • autoprefix, minify and purge the CSS bundle
  • compile any TS files to a JS (ES6) bundle file (no modules!)
  • uglify the JS bundle
  • optimizing any jpg/png images
